#技术栈
<https://github.com/lizhanlz/react-dianping>

react + react-router + redux + less + ES6/7 + webpack + fetch

#安装

  
	git clone https://github.com/lizhanlz/react-dianping  
  
	cd react-dianping  
  
	npm instal

#运行

	npm run mock (启动模拟后台服务器，提供api)
	
	npm run dev (启动前台程序)
访问 http://localhost:8080
